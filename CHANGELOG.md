## Version Scheme

Alternator follows **Early SemVer**:
- **Major version (x.0.0)**: Breaking changes to public API
- **Minor version (0.x.0)**: New features, deprecations (backward compatible)
- **Patch version (0.0.x)**: Bug fixes only (backward compatible)

Binary compatibility is maintained within minor versions.
